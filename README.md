# Reverse Engineering

## 项目简介
这是一个关于逆向工程的项目，旨在学习和实践逆向工程的基本原理和工具。

## 项目内容
- **工具**：使用 Ghidra、Binary Ninja、IDA Pro 等工具进行逆向分析。
- **目标**：分析二进制文件、破解加密算法、理解程序逻辑。
- **示例项目**：包含一些简单的二进制文件和练习。
